## Sidenotes

Numbered sidenote — marker in text, definition below the paragraph:[^1]

[^1]: This is the sidenote text. It appears numbered in the margin.

```
Numbered sidenote — marker in text, definition below the paragraph:[^1]

[^1]: This is the sidenote text.
```

---

Margin note — no number, no marker in text, appears inline:

[^ {-} This is an unnumbered margin note. It floats in the margin with no reference number.]

```
[^ {-} This is an unnumbered margin note.]
```

---

Margin note linked to a specific word:[^linked]

[^linked]: {-} This margin note is anchored to the superscript above.

```
Margin note linked to a specific word:[^linked]

[^linked]: {-} This margin note is anchored to the superscript above.
```

## Typography

<span class="newthought">New thought</span> — small caps, used to open a section. Raw HTML only, no markdown shorthand.

```html
<span class="newthought">New thought</span>
```

*Italic*, **bold**, `inline code` — standard markdown.

## Blockquotes and Epigraphs

Standard blockquote:

> The life of the law has not been logic; it has been experience.

```
> The life of the law has not been logic; it has been experience.
```

---

Epigraph with attribution — requires the `<div class="epigraph">` wrapper and a `<footer>` tag:

<div class="epigraph">

> To appreciate and use correctly a valuable maxim requires a genius.

<footer>Ralph Waldo Emerson</footer>
</div>

```html
<div class="epigraph">

> To appreciate and use correctly a valuable maxim requires a genius.

<footer>Ralph Waldo Emerson</footer>
</div>
```

## Images and Figures

Standard image — sits in the text column:

```
![Alt text](url)
```

---

Full-width figure — spans text column and margin:

```html
<figure class="fullwidth">

![Alt text](url)

<figcaption>Caption here.</figcaption>
</figure>
```

---

Margin figure — image and caption both in the margin. Raw HTML only:

```html
<figure class="marginfigure">
  <label for="mf-unique-id" class="margin-toggle">&#8853;</label>
  <input type="checkbox" id="mf-unique-id" class="margin-toggle"/>
  <span class="marginnote">
    <img src="url" alt="Alt text"/>
    Caption here.
  </span>
</figure>
```

## Known Gotchas

**UK law report citations** — `[2024] EWCA Civ 1` confuses the parser (year in square brackets reads as a footnote reference). Escape them:

```
\[2024\] EWCA Civ 1
```

**Links inside margin notes** — `[link text](url)` inside `[^ {-} ...]` breaks the parser. Use plain text in margin notes.

**Stylesheet order** — `main.css` must load *after* `tufte.min.css` or your overrides will be silently ignored. Set in `src/_includes/head.njk`.
